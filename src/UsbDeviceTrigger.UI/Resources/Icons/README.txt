Icône de l'application
======================

Pour ajouter une icône à l'application:

1. Créez ou téléchargez une icône .ico (format Windows Icon)
2. Nommez-la "app-icon.ico"
3. Placez-la dans ce dossier (Resources/Icons/)
4. L'icône apparaîtra:
   - Dans la barre des tâches
   - Dans le system tray
   - Dans le menu Démarrer
   - Dans la barre de titre de la fenêtre

Taille recommandée: 256x256 pixels avec plusieurs résolutions intégrées (16x16, 32x32, 48x48, 256x256)

Outils gratuits pour créer des icônes:
- IcoFX (https://icofx.ro/)
- GIMP (https://www.gimp.org/)
- Online Icon Converter (https://convertio.co/png-ico/)

Note: En attendant d'avoir une icône personnalisée, l'application utilisera l'icône par défaut Windows.
